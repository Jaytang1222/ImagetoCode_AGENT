# 📝 更新日志

## 2026.4.9

### ✨ 新增功能

1. **Flask 后端服务**
   - 将 main.py 修改为了 flask 后端形式

2. **前端页面**
   - 制作了初版前端页面 frontend/index.html

3. **历史记录查看**
   - 实现了在页面中查看历史记录
   - **解决方案**：在 main.py 中添加接口 `/api/outputs`，获取 outputs 目录信息作为历史记录，前端访问该接口，将后端的数据返回给前端，展示在页面中

4. **在线预览 Echarts**
   - 支持了在线预览 Echarts 功能
   - **解决方案**：
     - 在 main.py 中添加接口 `/api/preview/dir/<dir_name>`
     - 使用 iFrame 框架，在"查看" button 下添加 iFrame 标签，用上述接口，将后端的 Echarts 数据返回给前端，并动态展示在 iFrame 标签中

5. **进度条动态展示**
   - 实现了进度条动态展示功能
   - **遇到困难**：在调度器中仅当开始和结束时检查状态
   - **解决方案**：添加一个定时任务，每隔 2 秒检查一次状态，并更新进度条；在后端和前端增加相应的接口和接口调用
